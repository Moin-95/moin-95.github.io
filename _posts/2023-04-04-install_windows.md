# How to Install Windows: A Simplified Guide

Installing Windows can seem like a complicated task, but with a few simple steps, anyone can do it. Here's a guide to help you through the process.

## Things You'll Need:

1. A computer
2. A Windows installation ISO file
3. Rufus software (you can download it for free from the official website)
4. A USB drive (at least 8 GB)

## Step-by-Step Guide:

1. Download the Windows installation ISO file from Microsoft's website.
2. Download Rufus software and install it on your computer.
3. Insert your USB drive into your computer.
4. Open Rufus and select your USB drive under the "Device" dropdown menu.
5. Under the "Boot selection" section, click "Select" and browse for the Windows installation ISO file you downloaded.
6. Under the "Partition scheme" section, select MBR or GPT. If you're not sure, choose the default option.
7. Under the "File system" section, leave the default option (NTFS).
8. Click "Start" to begin the flashing process. Rufus will warn you that all data on the USB drive will be destroyed. Click "OK" to continue.
9. Wait for Rufus to finish flashing the Windows installation files onto the USB drive. This may take a few minutes.
10. After Rufus is finished, safely eject the USB drive from your computer.

### Boot from USB Drive

11. Turn on your computer and press the key that brings up the boot menu. This is usually F12, but it may be different for your computer. If you're not sure, look it up online.
12. Use the arrow keys to select the option to boot from the USB drive.
13. Wait for Windows to load the installation files. This may take a few minutes.

14. Select your language, time and currency format, and keyboard or input method, and then click "Next."
15. Click "Install now."
16. Read and accept the license terms, and then click "Next."
17. Choose "Custom: Install Windows only (advanced)."
18. Select the hard drive or partition where you want to install Windows, and then click "Next."
19. Wait for Windows to install. This may take a while, so be patient.
20. After Windows is installed, your computer will restart.

21. Follow the on-screen instructions to set up Windows. This includes things like creating a user account, setting up a password, and connecting to the internet.
22. Enter your product key when prompted.
